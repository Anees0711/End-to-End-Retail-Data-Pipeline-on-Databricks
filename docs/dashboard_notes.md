# Dashboard Notes

The Power BI dashboard for this project should focus on retail performance and business insights.

## Recommended dashboard sections

### 1. Executive Summary
- Total sales
- Total transactions
- Average order value
- Top category

### 2. Sales Trends
- Daily sales trend
- Sales by category

### 3. Product Insights
- Top products
- Category contribution
- Stock status distribution

### 4. Customer Insights
- Customer segments
- Country distribution
- Customer type breakdown

## Recommended source tables
- gold_daily_sales
- gold_category_sales
- silver_orders
- silver_products
- silver_customers
